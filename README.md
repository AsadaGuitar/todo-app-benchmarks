Todo application for benchmarking.<br>
Benchmarks are obtained for server counts 1-3.

## Feature
+ REST API
+ Rust ver.1.64.0
+ actix-web ver.4
+ AWS (EC2, ELB, RDS)
+ Docker

## Starting up node app
```terminal
# install git, docker
$ git clone https://github.com/AsadaGuitar/todo-app-benchmarks.git
$ cd todo-app-benchmarks
$ chmod 755 ./run.sh
$ ./run.sh
```
